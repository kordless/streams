Labs Documentation

Introduction to Labs

This page contains documention for Lucidwork's Labs, which has been designed to enable easy evalution of Lucidwork's products. Documentation for Lucidwork's Fusion line of products is located on the Lucidworks Documentation page.

Be sure to visit Lucidwork's Labs Streams Repo on Github for more code, scripts and video guides.

Features

Lucidworks Labs Supports the Following:

  • Quick starts of Fusion based instances
  • Instance timers
  • Other things

Signup

Lucidworks Labs uses Github for authentication. To login, please visit your Lab's Dashboard.

Instance API

A given instance's status may be retrieved by making a call to the Instance API. An instance status is retreived using the following method:

GET /api/instance/
curl https://streams.lucidworks.com/api/button-fusion-ewyp
